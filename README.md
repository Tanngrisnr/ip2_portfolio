# Fredrik Wintzell portfolio
[Live version](https://wintzell.surge.sh/)
	
This website was created to be a simple portfolio page. Showcasing knowledge in React, Javascript, as well as in working with APIs and Styled-Components. The site features a weather widget and a toggleable theme based on the current season.
